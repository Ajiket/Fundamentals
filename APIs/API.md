# API Fundamentals

## What is an API?
- Application Programming Interface
- Acts as a bridge between software systems
- Example: Weather app fetching data from a weather service

## What is an API Call?
- A request sent to an API endpoint
- Types: GET (retrieve), POST (create), PUT/PATCH (update), DELETE (remove)
- Example: `GET https://api.weather.com/v1/pune?date=2026-03-31`

## Why APIs Matter
- Enable integration
- Save development time
- Provide scalability
